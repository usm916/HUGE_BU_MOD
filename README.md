# HUGE_BU_MOD

エレコムHUGE用の人工ルビー支持球をベアリングユニットに置きかえるための3Dモデルを公開しています。
* HUGE_balltray_BU.stl :: ベアリングユニットを乗せるためのモデルです。
* HUGE_for_sensorPCB.stl :: センサー用のPCBを固定するためのモデルです。

二つのモデルを組み合わせて仕様します。<br>
